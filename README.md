### Bienvenid@s a Tu Archivo ZIP (v1.1.0)

##### **Compresor ZIP**

##### ✨ En esta versión se ha unificado todo en un solo archivo `.py`, más completo, interactivo y con mejor estructura.
> [!NOTE]
> ###### - `zip_manager.py` es ahora el único archivo principal.  
> ###### - Permite arrastrar archivos directamente a la terminal para comprimirlos.  
> ###### - Puedes comprimir uno o varios archivos a la vez.  
> ###### - Al finalizar, el programa pregunta si deseas volver a comprimir o salir.  

#### 🚀 Mejoras respecto a la versión anterior

- Unificación de los tres scripts en uno solo.
- Interfaz interactiva que permite repetir sin cerrar el programa.
- Limpieza de pantalla automática entre ejecuciones.
- Control de errores con mensajes claros.
- Validación de rutas y archivos inexistentes.
- Preparado para futuras mejoras (colores, logs, interfaz gráfica...).
